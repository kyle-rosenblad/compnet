# Species-level data for two traits in a simulated metacommunity dataset with 24 species

A data frame with 24 rows and 2 columns:

- ndtrait:

  A continuous trait driving competitive niche differentiation

- domtrait:

  A continuous trait influencing competitive ability

- ctrait:

  A categorical trait influencing competitive niche differentiation

## Usage

``` r
ex_traits
```

## Format

An object of class `data.frame` with 24 rows and 3 columns.
