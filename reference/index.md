# Package index

## All functions

- [`buildcompnet()`](https://kyle-rosenblad.github.io/compnet/reference/buildcompnet.md)
  : Network models of interspecific competition with presence-absence
  data
- [`compnet`](https://kyle-rosenblad.github.io/compnet/reference/compnet.md)
  [`compnet-package`](https://kyle-rosenblad.github.io/compnet/reference/compnet.md)
  : The 'compnet' package.
- [`compute_gofstats()`](https://kyle-rosenblad.github.io/compnet/reference/compute_gofstats.md)
  : Compute summary statistics quantifying row/column-level and third
  order dependencies in a symmetric matrix with NA diagonals
- [`ex_phylo`](https://kyle-rosenblad.github.io/compnet/reference/ex_phylo.md)
  : Phylogenetic distance between species pairs in a simulated
  metacommunity dataset with 24 species
- [`ex_presabs`](https://kyle-rosenblad.github.io/compnet/reference/ex_presabs.md)
  : A simulated metacommunity presence-absence matrix in which rows
  represent sites and columns represent species
- [`ex_traits`](https://kyle-rosenblad.github.io/compnet/reference/ex_traits.md)
  : Species-level data for two traits in a simulated metacommunity
  dataset with 24 species
- [`expit()`](https://kyle-rosenblad.github.io/compnet/reference/expit.md)
  : Inverse logit function
- [`fixedeff_violins()`](https://kyle-rosenblad.github.io/compnet/reference/fixedeff_violins.md)
  : Violin plots of fixed effect coefficients in a compnet model
- [`gofstats()`](https://kyle-rosenblad.github.io/compnet/reference/gofstats.md)
  : Report quantiles of gofstats values for observed data relative to
  posterior predictive distribution
- [`plot_interaction()`](https://kyle-rosenblad.github.io/compnet/reference/plot_interaction.md)
  : Make an automated plot of the interactive effect of two species'
  values of the same trait.
- [`plot_interaction_getdata()`](https://kyle-rosenblad.github.io/compnet/reference/plot_interaction_getdata.md)
  : Get data for a plot of the interactive effect of two species' values
  of the same trait.
- [`plot_pairvar()`](https://kyle-rosenblad.github.io/compnet/reference/plot_pairvar.md)
  : Make an automated plot of the effect of a pair-level variable like
  phylogenetic distance
- [`plot_pairvar_getdata()`](https://kyle-rosenblad.github.io/compnet/reference/plot_pairvar_getdata.md)
  : Get data for a plot of the effect of a pair-level variable like
  phylogenetic distance
- [`postpredsamp()`](https://kyle-rosenblad.github.io/compnet/reference/postpredsamp.md)
  : Simulate posterior predictive samples from a compnet model
- [`scatter_interaction()`](https://kyle-rosenblad.github.io/compnet/reference/scatter_interaction.md)
  : Deprecated. Please use plot_interaction().
- [`scatter_interaction_getdata()`](https://kyle-rosenblad.github.io/compnet/reference/scatter_interaction_getdata.md)
  : Deprecated. Please use plot_interaction_getdata().
- [`scatter_pairvar()`](https://kyle-rosenblad.github.io/compnet/reference/scatter_pairvar.md)
  : Deprecated. Please use plot_pairvar().
- [`scatter_pairvar_getdata()`](https://kyle-rosenblad.github.io/compnet/reference/scatter_pairvar_getdata.md)
  : Deprecated. Please use plot_pairvar_getdata().
- [`summarize_compnet()`](https://kyle-rosenblad.github.io/compnet/reference/summarize_compnet.md)
  : Summarizing compnet model output
