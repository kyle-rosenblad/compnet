# Phylogenetic distance between species pairs in a simulated metacommunity dataset with 24 species

A data frame with 276 rows and 3 columns:

- spAid:

  ID for species A in each pair

- spBid:

  ID for species B in each pair

- phylodist:

  Phylogenetic distance between each species pair

## Usage

``` r
ex_phylo
```

## Format

An object of class `data.frame` with 276 rows and 3 columns.
